# 🤖 Customer Support Chatbot

A simple AI-based Customer Support Chatbot built using Python and Streamlit.

## Features

- 👋 Greeting Response
- 📦 Order Status
- 💰 Refund Policy
- 🚚 Shipping Information
- 💳 Payment Methods
- ❓ FAQ Response
- 🔄 Fallback Response
- 🎨 Attractive User Interface

## Technologies Used

- Python
- Streamlit
- Pandas

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
streamlit run app.py
```
